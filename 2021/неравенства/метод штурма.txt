< # Метод Штурма

<div style="width:16cm; margin:auto;">

**Метод Штурма** обычно используется для доказательства неравенств с несколькими переменными.

Основная его идея заключается в следующем: менять одновременно две переменные, сохраняя их сумму (или произведение) так, чтобы при этом одна из частей неравенства всегда изменялась в одну сторону. Обычно переменные либо пытаются сделать равными, либо наоборот — раздвинуть как можно дальше друг от друга.

**Пример 1.** У Арсения было два положительных числа $a$ и $b$. Арсений заменил их на два новых числа $c$ и $d$ такие, что $a+b = c+d$, $|a-b| > |c-d|$. Как изменятся значения следующих выражений:
- $ab$
- $a^2+b^2$
- $\frac{1}{a} + \frac{1}{b}$
- $\sqrt{a} + \sqrt{b}$
- Пусть положительные числа удовлетворяют неравенству $a < c < d< b.$
Тогда если $ab = cd$, то $a + b > c + d$.

*Ответы:* 
$ab$ увеличится;
$a^2+b^2$ уменьшится;
$\frac{1}{a} + \frac{1}{b}$ увеличится; $\sqrt{a} + \sqrt{b}$ увеличится.

**Пример 2.** Докажите, что $(1 + x_1)(1 + x_2)\ldots (1 + x_n) \ge 2^n$, где $x_1 x_2 \ldots x_n = 1$, $x_i > 0$.

*Решение.* Если $x_1=x_2= \ldots = x_n=1$, тогда $(1 + x_1)(1 + x_2)\ldots (1 + x_n) = 2^n$. Пусть среди $x_1, x_2, ..., x_n$ есть хотябы два не равных. Тогда найдутся два числа, что одно больше 1, другое меньше. Пусть это $x_1>1, x_2 <1$. Тогда верно
$$(1+x_1)(1+x_2) \ge (1+1)(1+x_1x_2)$$
потому что
$$(x_1-1)(1-x_2) \ge 0.$$
Получается, можно заданные числа можно заменить на $1, x_1x_2, x_3, \ldots, x_n$, сохранить произвидение 1 и получить значение меньше
$$(1+x_1)(1+x_2) \ldots (1 + x_n)> (1+1)(1+x_1x_2)(1+x_3)\ldots (1 + x_n).$$
Действуя аналогично с числами $1, x_1x_2, x_3, \ldots, x_n$ получим числа среди которых хотя бы три равных. Действуя так же $n-1$ раз получим набор где $n-1$ чисел равны, а $n$-ое равно $x_1x_2 \ldots x_n$ т.е. будет 1.
Таким образом $$(1+x_1)(1+x_2) \ldots (1 + x_n)>2^n$$
и равенство достигается  тогда и только тогда, когда $x_1=x_2= \ldots = x_n=1$.

**Пример 3.** Докажите для положительных $x_1 , x_2 , \ldots , x_n$ неравенства между средними (слева направо: среднее гармоническое, геометрическое, арифметическое, квадратичное):
$$\frac{n}{\frac{1}{x_1}+\frac{1}{x_2}+\ldots+\frac{1}{x_n}} \le \sqrt[n]{x_1x_2 \ldots x_n} \le \frac{x_1+x_2+\ldots+x_n}{n} \le \sqrt{\frac{x_1^2+x_2^2+\ldots + x_n^2}{n}}$$
*Решение.* Можно доказать в лоб, но мы докажем вспомогательное неравество.
Рассмотрим положительные $y_1, y_2, \ldots, y_n$ произвидение которых 1. Тогда, верно
$$n \le y_1+y_2+\ldots+y_n.$$
При $y_1 = y_2 = \ldots = y_n = 1$ достигается равенство. Пусть нашлись два не равных числа среди них, они такие, что $y_1 > 1, y_2 < 1$. Получается верно $y_1 + y_2 \ge y_1y_2 +1$, так как $(y_1-1)(1-y_2)\ge0$. Значит, можно заменить данные числа на $1, y_1y_2, y_3, \ldots, y_n$ и получить сумму меньше. Аналогично повторим $n-1$ раз, тогда будет хотя бы $n-1$ равных и $n$-ый будет $y_1y_2\ldots y_n$ равный 1.
Значит $$n \le y_1+y_2+\ldots+y_n$$ и равенство только при $y_1 = y_2 = \ldots = y_n$. <span style="float:right;">$\square$</span>

Применив это неравество для $y_i = \frac{\sqrt[n]{x_1x_2 \ldots x_n}}{x_i}$ получим неравенство средне гармоничное-геометрическое. Аналогично применив это неравество для $y_i = \frac{x_i}{\sqrt[n]{x_1x_2 \ldots x_n}}$ получим неравенство средне геометрическое-арифмитическое. <span style="float:right;">$\square$</span>

Для средне арифметическое-квадратическое неравенства рассмотрим положительные числа $z_1, z_2, \ldots, z_n$ сумма которых равна 1. Тогда нужно доказать
$$\frac{1}{n} \le z_1^2+z_2^2 + \ldots z_n^2.$$
При $z_1=z_2=\ldots=z_n=\frac{1}{n}$ достигается равенство. Пусть нашлись два не равных $z_1, z_2$. Не трудно показать, что $z_1^2+z_2^2 \ge 2 (\frac{z_1+z_2}{2})^2$. Значит, данные числа можно заменить на $\frac{z_1+z_2}{2}, \frac{z_1+z_2}{2}, z_3, \ldots, z_n$, сохранить сумму и получить
$$(\frac{z_1+z_2}{2})^2+(\frac{z_1+z_2}{2})^2+z_3^2 + \ldots z_n^2 \le z_1^2+z_2^2 + \ldots +z_n^2.$$
Повторяя аналогично $n-1$ раз получим что когда все равны

$$\frac{1}{n} =n(\frac{z_1+z_2+\ldots+z_n}{n})^2 \le z_1^2+z_2^2 + \ldots + z_n^2.$$

Поставим $z_i = \frac{x_i}{x_1+x_2+...+x_n}$ и получим требуемое неравенство. <span style="float:right;">$\square$</span>
